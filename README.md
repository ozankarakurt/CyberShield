CyberShield 🛡️
CyberShield, siber güvenlik analiz süreçlerini hızlandırmak ve karmaşık güvenlik görevlerini modern bir arayüzle kolaylaştırmak için Python ve PyQt5 kullanılarak geliştirilmiş bir masaüstü uygulamasıdır.

🛠️ Temel Özellikler
Dashboard: Sistem durumunu ve güvenlik loglarını takip edebileceğiniz merkezi yönetim paneli.

URL Scanner: Şüpheli bağlantıları VirusTotal entegrasyonu ile tarayarak risk analizi yapar.

Password Analyzer: Parolalarınızın karmaşıklığını ve güvenlik düzeyini ölçer.

File Analyzer: Dosya bütünlüğünü (Hash değerleri üzerinden) kontrol eder.

Modern Arayüz: PyQt5 ile geliştirilmiş, şık, hızlı ve "Fusion" temalı kullanıcı deneyimi.

Cross-Platform: Hem Kali Linux hem de Windows işletim sistemlerinde sorunsuz çalışır.

🚀 Kurulum (Installation)
Projeyi bilgisayarınızda çalıştırmak için şu adımları izleyin:

1. Ön Gereksinimler
Bilgisayarınızda Python'ın yüklü olduğundan emin olun. python.org adresinden indirebilirsiniz.

2. Kütüphaneleri Yükleyin
Terminali (veya Komut İstemi'ni) açın ve gerekli bağımlılığı kurun:

Bash
pip install PyQt5
3. Uygulamayı Başlatın
Uygulamanın bulunduğu klasöre gidin ve terminalde şu komutu çalıştırın:

Bash
python "CyberSHIELD.py"
